# Projeto Unity - Mapa DayZ em 3D

Este projeto traz uma recriação do mapa de DayZ em Unity, com foco em melhorias gráficas, otimização de FPS e experiência imersiva em 3D.

## Ilustração

![Mapa DayZ em 3D](https://upload.wikimedia.org/wikipedia/commons/thumb/5/5e/DayZ_Standalone_Screenshot.jpg/640px-DayZ_Standalone_Screenshot.jpg)

## Funcionalidades
- Recriação fiel do mapa original de DayZ
- Otimizações gráficas para melhor desempenho
- Ajustes de iluminação e texturas
- Suporte a mods de FPS e frame rate

## Como usar
1. Clone este repositório
2. Abra o projeto no Unity
3. Compile e execute para explorar o mapa em 3D
